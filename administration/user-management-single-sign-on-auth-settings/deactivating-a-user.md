# Deactivating a User

You can **deactivate** the existing **user accounts** that **you do not want to log into** Oobeya.

_Deactivated users_ can not log into the Oobeya.

There are _two_ options for this operation:

### **1. Deactivating a user from the user list:**

1. Navigate to _**Administration Panel > Admin Settings**_.
2. Open _**Users & Groups**_.
3. Click the **settings icon** of the user you want to deactivate and click **"Inactive"**.&#x20;

![](https://firebasestorage.googleapis.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MGIlBSTjQtZxUoFwUx4%2Fuploads%2FaIh7KyyKAjitLnrrX8Dd%2Ffile.png?alt=media)

### **2. Deactivating a user from the user detail:**

1. Navigate to _**Administration Panel > Admin Settings**_.
2. Open _**Users & Groups**_.
3. Click the **user** you want to deactivate
4. Select the _Status_ as "_**Deactive**_" and _Save_.&#x20;
