#### 2. Click on Go to Admin Settings

#### 3. Open the "Gitwiser" tab

<figure><img src="https://images.tango.us/workflows/1150dd22-492d-4f31-8d22-6f1da0ad7958/steps/ae0dc270-faa6-48aa-a8c7-9c620761701c/d1370d35-b871-41e8-adfc-c50cb41859a3.png?fm=png&#x26;crop=focalpoint&#x26;fit=crop&#x26;fp-x=0.1063&#x26;fp-y=0.3451&#x26;fp-z=2.9165&#x26;w=1200&#x26;mark-w=0.2&#x26;mark-pad=0&#x26;mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&#x26;ar=3584%3A1866" alt=""><figcaption></figcaption></figure>

#### 4. Enable toggle to set a reanalyze period for the Gitwiser analysis.

If set to enabled, Gitwiser checks repositories periodically. Reanalyze processes are performed automatically if there are new commits, pull requests, and deployments on checked repositories.

<figure><img src="https://images.tango.us/workflows/1150dd22-492d-4f31-8d22-6f1da0ad7958/steps/dabf00b9-20c4-4f32-8ee5-494f42d07c75/bf74d29f-74f4-47b6-ad6a-b170965cf0a9.png?fm=png&#x26;crop=focalpoint&#x26;fit=crop&#x26;fp-x=0.4430&#x26;fp-y=0.4431&#x26;fp-z=1.1284&#x26;w=1200&#x26;mark-w=0.2&#x26;mark-pad=0&#x26;mark64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL21hZGUtd2l0aC10YW5nby13YXRlcm1hcmsucG5n&#x26;ar=3584%3A1866" alt=""><figcaption></figcaption></figure>

#### 5. Select a reanalyze period from the list.

Gitwiser checks and analyzes repositories at the following times if there are new commits, pull requests, and deployments:

* 15 Minutes: \*:00, \*:15, \*:30, \*:45
* 30 Minutes: \*:00, \*:30
* 1 Hour: \*:00
* 6 Hours: 00:00, 06:00, 12:00, 18:00
* 12 Hours: 00:00, 12:00
* 24 Hours: 00:00
