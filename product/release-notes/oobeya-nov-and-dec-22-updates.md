# 🎁 Oobeya Nov & Dec '22 Updates

## SUMMARY

&#x20;:tada: We are super excited to share our new features and improvements with you!

* [x] **DORA** - **Azure DevOps** [**Pipelines**](https://azure.microsoft.com/en-us/products/devops/pipelines/) integration is now available in Oobeya Deployment Analytics! You can use this feature to track DORA metrics in an accurate and effortless way. With this feature, Oobeya now supports both Azure Pipelines and Releases!
* [x] **DORA** - Oobeya Deployment Analytics now has **a new automated failure detection** feature that allows you to detect failures **by Git branch** **and** **Deployment title naming patterns**!
* [x] **DORA** - Oobeya Deployment Analytics now has **a new automated failure detection** feature that allows you to detect failures **by** [**New Relic**](https://newrelic.com/platform/alerts) **Incidents & Alerts**!
* [x] Gitwiser Git Analytics now allows you to **filter results by Team**!
* [x] We started to display the **historical timeline** of the Efficient Development Ratio!
* [x] In the Team Scorecard configuration, we've added a new suggestion feature that automatically selects deployments based on the selected repositories.
* [x] The Gitwiser module now displays the icon of the analyzed VCS add-on.
* [x] The [**Testinium**](https://testinium.com) **Add-on** has two new widgets: Success Rate & Average Runtime.
