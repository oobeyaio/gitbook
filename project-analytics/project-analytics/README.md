# Project Analytics

## 1. **Introduction**

Agile Space thoroughly analyzes an Agile Team's entire working practices. It draws attention to bottlenecks and problematic units for each step (state, status) and individual.

* Agile Space provides metrics that guide the team and PO (Product Owner) in Sprint Planning.
* Agile Space - provides detailed metrics to all stakeholders in the Sprint Review.
