# Board Overview

There are several widgets that users can view different metrics:

1. _**Summary:**_ The basic view of your board such as board name, board type, etc.&#x20;
2. _**Time in State:**_ It represents the average time a task stays in each state before completion.
3. _**Sprint Velocity Metrics:**_ It shows the velocity measurements of the amount of work a team can tackle during a single Sprint.
4. _**Assignee List:**_ It displays all assignees and their activities on the Board.
5. _**Work In Progress:**_ It represents the open issues that have been in progress for more than 5 days.
6. _**Scope Change:**_ It shows the changes during the sprint as work item/ticket count and estimated effort.
7. _**Estimated/Actual Effort:**_ It depicts the time that your team spends working on issues.
